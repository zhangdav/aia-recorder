### Build & Deploy

1. Install dependencies by running:

    ```bash
    yarn install
    ```

2. Regenerate code:

    ```bash
    yarn codegen
    ```

3. Create subgraph:

    ```bash
    yarn create-local
    ```

4. Deploy subgraph:

    ```bash
    yarn deploy-local
    ```
